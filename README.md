# 🌡️ Temperature Converter (VB.NET)
A simple Windows Forms application built using Visual Basic .NET that converts temperatures between Celsius and Fahrenheit.

## 🚀 Features
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Simple and user-friendly interface
- Input validation
- Exit button with confirmation

## 🛠️ Built With
- Visual Studio
- VB.NET (Windows Forms)
- .NET Framework

## 📂 How to Run

1. Open the project in Visual Studio
2. Build the solution
3. Run the application (Press F5)
4. Enter a temperature
5. Select conversion type
6. Click Convert

## 📌 Formula Used
- Celsius to Fahrenheit:  
  `(C × 9/5) + 32`

- Fahrenheit to Celsius:  
  `(F − 32) × 5/9`
